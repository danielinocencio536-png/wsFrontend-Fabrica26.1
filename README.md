# 🧪 Rick and Morty - Fábrica de Software 26.1

Este projeto é uma enciclopédia interativa de personagens da série **Rick and Morty**, desenvolvido como parte do desafio técnico da Fábrica de Software.

---

## 🚀 Funcionalidades

* 📋 **Listagem Dinâmica:** Exibição de personagens consumindo a API oficial
* 🔍 **Busca em Tempo Real:** Filtragem de personagens por nome
* 🌎 **Tradução Automática:** Status e espécies traduzidos para o português
* 📱 **Interface Responsiva:** Adaptável para mobile e desktop
* 🎨 **Design Temático:** Baseado na identidade visual da série

---

## 🛠️ Tecnologias Utilizadas

* Next.js (App Router)
* React
* Tailwind CSS
* Material UI (MUI)
* Axios

---

## 🌐 API Utilizada

Rick and Morty API
https://rickandmortyapi.com/

---

## ⚙️ Como rodar o projeto

```bash
npm install
npm run dev
```
